# Casa do Sorvete B2B

Plataforma B2B oficial da distribuidora **Casa do Sorvete**, desenvolvida para apresentar nossos produtos, vantagens de revenda e facilitar o contato com parceiros comerciais e revendedores.

## Tecnologias Utilizadas

O projeto foi construído utilizando as mais modernas tecnologias de desenvolvimento web:

- **[Next.js](https://nextjs.org/)**: Framework React para aplicações web rápidas e otimizadas.
- **[React](https://react.dev/)**: Biblioteca JavaScript para criação de interfaces de usuário interativas.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para estilização rápida e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca de animações para React, garantindo uma experiência de usuário fluida.
- **[Lucide React](https://lucide.dev/)**: Biblioteca de ícones bonitos e consistentes.

## Como Executar Localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/casadosorvete.git
   ```

2. Entre no diretório do projeto:
   ```bash
   cd casadosorvete
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver a aplicação.
